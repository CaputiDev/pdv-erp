# Welcome to your Expo app 👋

This project was initialized using Expo. To get started, follow the steps below.

## 🚀 Getting Started

1. **Install dependencies** (if not already installed):
   ```bash
   pnpm install
   ```

2. **Start the development server**:
   ```bash
   npx expo start
   ```

## 📱 Running the App

Once the server is running, you can open your app in several ways:

* **Expo Go:** Scan the QR code displayed in your terminal with the Expo Go app on your physical iOS or Android device.
* **iOS Simulator:** Press `i` in your terminal (requires Xcode installed on macOS).
* **Android Emulator:** Press `a` in your terminal (requires Android Studio installed).
* **Web:** Press `w` in your terminal to open the app in your browser.

## 🛠️ Making Changes

* Open `app/index.tsx` (or your entry file) in your code editor.
* Make changes to the code and save the file.
* The app will automatically reload to reflect your changes (Fast Refresh).
