Este repositório contém a solução completa para o **PDV ERP**, um sistema prático e dinâmico voltado para pequenos comerciantes, lojistas e autônomos gerenciarem estoques, clientes e vendas de forma integrada e offline-first.

---

## 📁 Estrutura do Projeto

O projeto é dividido nas seguintes frentes:

*   **`client/` (Frontend)**: Aplicativo mobile construído com **React Native, Expo, TypeScript e NativeWind (Tailwind CSS)**. Focado em velocidade operacional, contendo o fluxo de dashboard, catálogo de produtos, cadastro de clientes e fluxo animado de finalização de pedidos localmente.
*   **`server/` (Backend)**: API e banco de dados responsáveis pelo sincronismo de dados em nuvem, controle centralizado e gestão persistente.
